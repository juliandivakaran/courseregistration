Course Registration System ReadMe

Website URL: https://github.com/juliandivakaran/courseregistration/

---

Description:
This is a simple Course Registration System website developed using HTML, CSS, JavaScript, and Bootstrap. The system allows users to register for courses, login to their accounts, view course information, and more.

---

Directory Structure:
- Source_Code/
  - index.html
  - registration.html
  - login.html
  - course_info.html
  - course_reg.html
  - registration.js
  - login.js
  - course_reg.js
  - style.css
- ReadMe.txt

---

Usage:
1. To use the website, simply open the provided HTML files in a web browser.
2. Navigate through the website using the links provided in the navigation bar.
3. Fill out the registration form to sign up for a new account.
4. Use the login form to log in to your account.
5. Explore the course information page to learn about available courses.
6. Register for courses using the course registration form.
